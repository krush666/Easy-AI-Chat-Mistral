Easy AI Chat Embed can integrate the powerful *Mistral AI* chat agent into your posts and pages.

== Description ==

Transform your WordPress site into an interactive hub with the Easy AI Chat Embed plugin. Fully supports custom Mistral AI agents and Compatible with the Mistral API Tiers, seamlessly integrate the powerful Mistral AI Chatbot into your posts and pages, providing your visitors with instant, intelligent responses. 
Whether you want to enhance customer support, engage users with interactive content, or provide quick answers to common questions, this plugin has you covered.

**Key Features:**
- **Easy Integration:** Embed the chatbot effortlessly using a simple short code.
- **Security** Rate limiting design allows 50 messages an hour per user.
- **Customizable Appearance:** Personalize the chat box and button color to match your site's design.
- **API Key Management:** Securely manage your Mistral AI API key directly from the settings page.
- **Dynamic Responses:** Leverage the advanced capabilities of Mistral AI to provide accurate and context-aware responses to user queries.
- **Model Selection:** Choose between different Mistral AI models to customize your chat experience.

**Why Choose Easy AI Chatbot Embed?**
- **Boost Engagement:** Keep your visitors engaged with real-time, interactive conversations. Or customize your agent to be your company receptionist.
- **Improve Customer Support:** Provide instant support and answers to common questions, reducing the load on your support team.
- **Enhance User Experience:** Offer a modern, AI-driven interface that enhances the overall user experience on your site.
- **Easy to Use:** With a user-friendly settings page, you can configure and customize the chatbot without any coding knowledge.

== Installation ==

1. Upload and install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Settings' -> 'Easy AI Chat' to configure the plugin settings.
4. Use the `[easy_ai_chat]` short code to embed the chatbot in any post or page.

== Frequently Asked Questions ==

= How do I get a Mistral AI API key? =
You can obtain a Mistral AI API key by signing up for the Mistral AI service on their official website.

= How do I customize the chat box and button color? =
Go to 'Settings' -> 'Easy AI Chat' and enter your desired colors in the color picker.

= How do I select a Mistral AI model? =
Go to 'Settings' -> 'Easy AI Chat' and select your preferred Mistral AI model from the dropdown menu. Currently supported models:
- Mistral Small (Latest)
- Mistral Small 2501
- Open Mistral Nemo
- Codestral 2501
- Mistral Large 2411
- Open Mixtral 8x7B

= What are the differences between these models? =
- **Mistral Small (Latest)**: A lightweight, general-purpose model suitable for most chat applications.
- **Mistral Small 2501**: An earlier version of the small model.
- **Open Mistral Nemo**: An open-source model with broad capabilities.
- **Codestral 2501**: Specialized for code-related tasks and programming assistance.
- **Mistral Large 2411**: A more advanced, larger model with enhanced reasoning capabilities.
- **Open Mixtral 8x7B**: A powerful open-source model with a mixture of experts architecture.
